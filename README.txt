Disponibilizamos:

	- Uma pasta (win10-x64) com o build do projeto que possui um executável (trab.exe);

	- O código fonte feito em C#;

	- Relatório.pdf

	- Os 4 programas em Assembly e um txt comentado a respeito da lógica deles;



Para rodar o programa (no VSCode por exemplo) deve-se:

	- instalar o .NET Core 3.1 ---> 'https://dotnet.microsoft.com/download';

	- instalar a extensão 'C#' no VSCode;

	- use o comando 'dotnet run' para rodar.
